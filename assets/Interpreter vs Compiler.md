### 1. Interpreter

- 장점:  중간코드(기계어) 필요x, 개발 및 디버깅 빠름
- 단점:  실행 시마다 인터프리터 동작, 최적화 제한
- Python, JavaScript

### 2. Compiler

- 장점:  컴파일 과정에서 오류 발견으로 실행 시간에 발생하는 오류 감소
- 단점:  수정할 때마다 재컴파일(앱을 재배포(업데이트)), 최적화(최적의 실행코드)
- C++, Java

|  | 인터프리터 | 컴파일러 |
| :-------: | :-------: | :-------: |
| 번역 단위 | 전체 | 행(줄) |
| 번역 속도 | 빠름 | 느림 |
| 실행 속도 | 느림 | 빠름 |
